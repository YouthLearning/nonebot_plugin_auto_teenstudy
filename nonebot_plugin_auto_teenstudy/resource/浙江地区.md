## 设置指令

- ```
  添加大学习|set_dxx
  指令格式：添加大学习#地区#json格式用户信息
  ```

- ```
  设置大学习配置|添加大学习用户|add_dxx
  指令格式：设置大学习配置#QQ号#地区#json格式用户信息
  ```
## json格式用户信息表
- ```py
  {
  "name":"姓名",
  "openid":"自行抓取openid",
  "university":"学校名称",
  "college":"学院名称",
  "class_name":"团支部名称"
  }
